gdas.epower.check_filtering_settings¶

gdas.epower.check_filtering_settings(sample_rate, channels, tile_bandwidth, fmin, fmax)[source]¶

Check filtering settings and define the total number of channels and bandwidth to use for filter bank.

Parameters:

sample_rate : float

Sampling rate in Hz of the data retrieved from the metadata

min_frequency : float

Lowest frequency of the filter bank

max_frequency : float

Highest frequency of the filter bank

channels : int

Number of frequency channels to use

tile_bandwidth : float

Bandwidth of the finest filters
